Le développement WEB

Contexte :
Le développement WEB est un terme bien trop large pour réellement décrire toute la puissance du langage HTML, CSS, JS, PHP et bien d'autres. Ces derniers permettent de développer, premièrement, des interfaces WEB, comme ce site, conçu en HTML (la structure du site), CSS (le style du site),
et PHP (le "comportement" du site). Toutes ces imbrications permettent d'arriver à un résultat présentable sur Internet : il s'agit du développement WEB.
